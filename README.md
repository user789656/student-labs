##  Запуск
1. Создать виртуальное окружение
```bash
python -m venv venv
```
3. Активировать виртуальное окружение
```bash
source venv/Scripts/activate
```
5. Создать файл .env и ввести свои данные от телеграмм:
```
TOKEN = ...
BOSS_CHAT_ID = ...
```
5. Запустить код:
```bash
main.py
```
