##  Запуск
1. Создать виртуальное окружение
2. Активировать виртуальное оокружение
3. Создать файл .env и ввести свои данные от телеграмм:
```
TOKEN = ...
BOSS_CHAT_ID = ...
```
5. Запустить main.py
